# Quickstart

Get started with TaskAI in under 5 minutes.

## 1. Create Your Account

Sign up at [taskai.cc/signup](https://taskai.cc/signup) with your email or sign in with Google or GitHub.

## 2. Create a Project

After logging in, click **New Project** and give it a name and description. Your project comes with default swim lanes: **To Do**, **In Progress**, and **Done**.

## 3. Add Tasks

Click **Add Task** on any swim lane to create a task. Set the title, description, priority, and assign it to a team member.

## 4. Invite Your Team

Go to **Project Settings** → **Members** and invite teammates by email. They'll receive an invitation to join your project.

## 5. Connect AI Agents (Optional)

TaskAI includes a built-in MCP server so AI agents like Claude can manage your tasks directly. See the [MCP Integration guide](/mcp/overview) to set it up.
